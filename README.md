# Reto-Ods4-WebSostenible
## El "Bug" Educativo

> En muchos centros educativos los alumnos **no tienen acceso fácil a materiales digitales**.  
> Además, los recursos suelen estar **dispersos en diferentes plataformas**, lo que dificulta
> encontrar apuntes, tareas o información importante.
---
## Nuestro "Parche" Sostenible
La aplicación web buscará mejorar el acceso a la educación digital siguiendo el **ODS 4**.

### Medidas sostenibles

- [ ] **Plataforma ligera:** diseñada para consumir **pocos datos** y cargar rápido.
- [ ] **Accesibilidad:** compatible con **lectores de pantalla** para personas con discapacidad visual.
- [ ] **Modo oscuro:** reduce el **consumo de batería** en dispositivos móviles.
- [ ] **Material centralizado:** todos los recursos educativos estarán en **un solo lugar**.
